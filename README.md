# Hotel Reservation System

## 📌 Project Description
Hotel Reservation System is a Java console application that allows users to search available rooms, book rooms, cancel reservations, and manage hotel bookings.

## 🚀 Features
- View available rooms
- Room categories (Standard, Deluxe, Suite)
- Book rooms
- Cancel bookings
- Payment simulation
- View booked rooms

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## ▶️ How to Run
1. Open the project in your Java IDE.
2. Compile the program.
3. Run `HotelReservationSystem.java`.

## 📷 Sample Output
Displays room availability, booking confirmation, cancellation, and booking details.
